# **App Name**: Horizon Bank

## Core Features:

- Secure Account Access: Robust user authentication with secure login options, including multi-factor authentication (MFA).
- Personal Account Overview: View balances and transaction history for checking and savings accounts in an intuitive dashboard.
- P2P Transfers: Facilitate peer-to-peer money transfers to other users, enhancing digital payment capabilities.
- Card Management: Instantly freeze and unfreeze virtual or physical debit cards to control spending and security.
- AI Financial Assistant: An AI-powered tool providing personalized financial insights, budget recommendations, and answering user queries regarding their accounts.
- Streamlined Onboarding: A guided digital onboarding process for new users to quickly and securely open a bank account.

## Style Guidelines:

- The primary brand color, conveying trust and innovation, is a deep, tranquil blue (#296EA4).
- The background will feature a serene, highly desaturated light blue (#E7EEF3) to ensure clarity and calm.
- A vibrant, forward-looking cyan (#4ACACA) serves as the accent color for key interactive elements and calls to action.
- All text, including headlines and body, will use 'Inter' (sans-serif), chosen for its modern, clear, and objective readability, vital for financial clarity.
- A minimalist, line-based icon set will be used, designed for clarity and a clean aesthetic, reinforcing the bank's digital-first approach.
- The layout will be clean and intuitive, utilizing card-based designs for easy navigation and digest of financial information, ensuring a seamless user experience.
- Subtle and purposeful micro-animations will be implemented for feedback on transactions, successful logins, and smooth transitions, enhancing user confidence and engagement without distraction.
